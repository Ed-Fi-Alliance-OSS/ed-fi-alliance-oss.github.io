# Ed-Fi Credentialing

Ed-Fi has two different types of credentials, Certifications and Badges.

## Certifications

Ed-Fi Certification allows:

* Product developers to demonstrate a product's fidelity to the Ed-Fi Data Standard and guidelines
* Purchasers and users to be confident that a product conforms to the same standard and guidelines

The program's goal is to ensure that systems can interoperate and exchange data using the Ed-Fi Data Standard.

Note: Certification is a test of the formal standard, but it also integrates and tests against best practice.

[Learn more about Ed-Fi Certification](/partners/certification/) and the [Best Practices](/partners/certification/certification-for-data-providers/api-integration-best-practices)

## Badges

Ed-Fi Badges allow product developers to demonstrate support for the Ed-Fi Data Standard and Ed-Fi Technology Suite, particularly in areas that are not yet covered by an Ed-Fi Certification.
[Learn more about Ed-Fi Badges](/partners/badging/).
