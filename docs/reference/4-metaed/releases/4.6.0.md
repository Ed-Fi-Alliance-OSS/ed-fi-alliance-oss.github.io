# 4.6.0

Release date: 14 November 2025

:::tip

If you already have the Visual Studio Code based editor, then it will
automatically update the MetaEd extension for you. If you are still using the
deprecated Atom based editor, then please see [MetaEd IDE - Upgrading to MetaEd
4.x](../ide-user-guide/upgrading-to-metaed-4x.md).

:::

## Overview

This release supports version 6.0 of the Ed-Fi Data Standard, and Ed-Fi ODS/API
version 7.3. It also removes support for obsolete versions of the Ed-Fi Data
Standard and ODS/API, in alignment with the Ed-Fi Technology Suite Support
Matrix.

## Changes

* MetaEd is now open source.
* Added Support for Data Standard 6.0.
* Removed support for older ODS/API and Data Standard versions that are no
  longer maintained.
