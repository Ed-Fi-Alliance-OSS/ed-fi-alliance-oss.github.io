# End of Support - Admin App v2.x

June 19, 2023

## What Is Happening

As of June 30, 2023, the Ed-Fi Admin App 2.x will have reached the [end of
support](./readme.md).

for the next school year; see [Ed-Fi Technology Suite Supported

for related updates.
