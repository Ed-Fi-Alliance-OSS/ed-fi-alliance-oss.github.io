# Legacy EPDM Developer Resources

:::warning Legacy Content

This page documents resources for the **EPDM extension** used prior to Ed-Fi Data
Standard v6. Starting with **Data Standard v6 (November 2025)**, EPDM is integrated into
the Ed-Fi core standard. See the
[Migration Guide](../migration-guide-epdm-to-ds6.md) for transition guidance.

:::

This page provides links to documentation covering the concepts and techniques necessary
for a technical professional to understand and successfully implement the Ed-Fi ODS/API
with Educator Preparation extensions.

## Audience

These resources are for current ODS/API with EPDM implementers.
Implementers are generally technical professionals who work with educational database
systems, including database administrators and software developers.

## Resources

- [EPDM-Community Model Updates](./epdm-community/model-updates.md) — Updates in the
  latest EPDM release.
- [EPDM UML Diagram](https://github.com/Ed-Fi-Exchange-OSS/Ed-Fi-TPDM-Community-Artifacts)
  — UML Entity-Relationship Diagram for EPDM entities.
- [Ed-Fi Data Model with EPDM Data Handbook](https://edfidocs.blob.core.windows.net/$web/handbook/tpdm-community-v1.2/Index.html)
  — Searchable, filterable EPDM-enhanced version of the Ed-Fi Data Handbook.
