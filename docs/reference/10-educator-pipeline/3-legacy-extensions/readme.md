# Legacy EPDM Extension Versions

:::warning Legacy Content

The pages in this section document the **EPDM extension versions** that existed prior to
Ed-Fi Data Standard v6. Starting with **Data Standard v6 (November 2025)**, the Educator
Preparation Data Model is fully integrated into the Ed-Fi core standard and is no longer
a separate extension.

For current documentation, see:

- [V6 Ed-Fi Data Standard EPDM Related Domains](../v6-epdm-related-domains.md)
- [Migration Guide: EPDM to DS 6.0](../migration-guide-epdm-to-ds6.md)

:::

## Background

Before Data Standard v6, EPDM was distributed as two separate ODS/API extensions:

- **EPDM Community** — The full-featured extension supporting broad use cases, including
  statewide implementations. Versioned independently from the core ODS/API.
- **EPDM Core** — A streamlined subset of foundational entities bundled directly with the
  ODS/API platform.

These extension versions are preserved here for reference by organizations maintaining
older implementations or performing historical comparisons.
