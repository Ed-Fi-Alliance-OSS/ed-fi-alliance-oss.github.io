# Legacy EPDM Community

:::warning Legacy Content

This page documents **EPDM Community**, which was an extension of the Ed-Fi ODS/API.
Starting with **Data Standard v6 (November 2025)**, EPDM is integrated into the Ed-Fi
core standard and no longer requires a separate extension. See the
[Migration Guide](../../migration-guide-epdm-to-ds6.md) for transition guidance.

:::

The Educator Preparation Data Model (Community version) (EPDM-Community) captures data
that allows an Educator Preparation Provider (EPP) to understand and answer key questions
relating to educator candidate development.

EPDM-Community includes every domain in EPDM-Core, plus:

- Expanded candidate information (indicators, characteristics, citizenship, background
  checks, and more)
- Recruitment and Staffing (applicants, applications, prospects, etc.)
- Credential library, certification exams, certification exam results, and certification
  events
- Professional development
- Non-rubric evaluations (quantitative measures, survey response scores, goals)
- Path Domain
