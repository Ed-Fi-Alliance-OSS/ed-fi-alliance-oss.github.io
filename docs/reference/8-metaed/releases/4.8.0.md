# 4.8.0

Release date: 9 July 2026

:::tip

By default, the Visual Studio Code based editor will automatically update the
MetaEd extension for you. If you are still using the deprecated Atom based
editor, then please see [MetaEd IDE - Upgrading to MetaEd
4.x](../ide-user-guide/upgrading-to-metaed-4x.md).

:::

## Overview

This release adds support for Ed-Fi API v8.0. It continues to support all
currently supported versions of the Ed-Fi Data Standard and ODS/API. See the
[Supported Versions](/reference/roadmap/supported-versions) page for the current
list.

## Changes

### Enhancements

* Added support for Ed-Fi API v8.0, including ApiSchema NuGet packaging as
  asset-only content packages for use by the Ed-Fi API.

## Bug Fixes

* Restored support for deploying to ODS/API 6.x, which had been broken when the
  deploy package moved to a 7.x-only directory structure.
