# Special Education Data Model Domain

The Special Education Data Model represents key events in a student's special education journey. It captures IDEA-related events within an education organization and tracks associated Individualized Education Programs (IEPs), including the services provided, how they are delivered, and when they occur to support the student's educational goals.
