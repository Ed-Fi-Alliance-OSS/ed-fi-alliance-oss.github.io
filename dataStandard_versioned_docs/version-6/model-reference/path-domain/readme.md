# Path Domain

The Path for a student to become an educator comprises many steps, including a program sequence of courses offered by the EPP, various assessments, fieldwork in a school, certification exams typically offered by a third party, and certifications from the state education agency. Paths are typically generic Paths to which many students are assigned. Different Paths may be defined for different pathways to certification. In addition, a Path may be customized for a specific student and is identified by a Path Name and reference to an Education Organization
