# What's New

This documentation serves as an introduction to the features and significant
changes made for the Ed-Fi ODS / API for Suite 3 v7.1.

This release is targeted at deployment starting in the 2024-25 school year. The
Ed-Fi ODS / API for Suite 3 v7.1 is aligned with [Ed-Fi Data Standard
v5.0](https://edfi.atlassian.net/wiki/spaces/EFDS5/overview), which introduced breaking changes to the core data model for
[domains](https://edfi.atlassian.net/wiki/display/EFDS5/What%27s+New+-+v5.0) commonly used by student information systems and assessment systems. Platform hosts should also be aware that there are changes to build and
deployment pipelines. In addition, this release requires implementers to update to [MetaEd
4.3](https://edfi.atlassian.net/wiki/spaces/METAED20/overview)

See [What's New in v7.1](./whats-new-in-this-release.md) for details.

## Audience

This documentation is for technical professionals who work with educational
data, including business analysts, database administrators, and
software developers.
