# What's New

This documentation serves as an introduction to the features and significant changes made for the Ed-Fi ODS / API for Suite 3 v5.4.

This release is aligned to Ed-Fi Data Standard v3.3.1-b 
([Ed-Fi Data Standard v3.3.1-b](https://edfi.atlassian.net/wiki/spaces/EFDS33)). 
There are no breaking changes between Ed-Fi ODS / API for Suite 3 v5.4 and the 
previous Ed-Fi ODS / API for Suite 3 v5.3. See [What's New in v5.4](./whats-new-in-v5.4.md) for details.

## Audience

This documentation is for technical professionals who work with educational data, including business analysts, database administrators, and software developers.
