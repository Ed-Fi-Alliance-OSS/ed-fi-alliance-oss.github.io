# What's New

This documentation serves as an introduction to the features and significant
changes made for the Ed-Fi ODS / API for Suite 3 v7.3.

This release is targeted at deployment starting in the 2025-26 school year. The
Ed-Fi ODS / API for Suite 3 v7.3 is aligned with [Data Standard
v5.2](https://edfi.atlassian.net/wiki/spaces/EFDS5/pages/26706990/What%27s+New).
There are no breaking changes between Data Standard v5.2 and the previous v5.x
Data Standard releases. Additionally, this release requires implementers to
update to [MetaEd
4.5](https://edfi.atlassian.net/wiki/spaces/METAED20/pages/23711107/What%27s+New)

See [What's New In This Release](./whats-new-in-this-release.md) for details.

## Audience

This documentation is for technical professionals who work with educational
data, including business analysts, database administrators, and software
developers.
