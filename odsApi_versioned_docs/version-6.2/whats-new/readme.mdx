# What's New

This documentation serves as an introduction to the features and significant
changes made for the Ed-Fi ODS / API for Suite 3 v6.2.

This release is targeted at deployment starting in the 2024-25 school year. It
is aligned to
[Ed-Fi Data Standard v4.0](/reference/data-exchange/data-standard/4/). There are
no breaking changes between Ed-Fi ODS / API for Suite 3 v6.2 and the previous
Ed-Fi ODS / API for Suite 3 v6.1.

See [What's New In This Release](./whats-new-in-this-release.md) for details.

## Audience

This documentation is for technical professionals who work with educational
data, including business analysts, database administrators, and software
developers.
