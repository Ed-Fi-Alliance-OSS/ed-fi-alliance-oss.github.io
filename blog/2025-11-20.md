## ODS/API 7.3.1

The ODS/API 7.3.1 includes support for Data Standard 6.0, along with Data
Standard 5.2 and 4.0. Key updates include the addition of domain metadata to the
OpenAPI spec and new configuration options to show or hide domain information in
swagger and to exclude unused domains for improved readability. This release is
intended to be **non-breaking**, enabling a smooth upgrade path from 7.3 to 7.3.1
for the next school year. See [What's New in this
Release](/reference/ods-api/whats-new/whats-new-in-this-release) for detailed
information.

## MetaEd 4.6

This is the first MetaEd release since the project became open source earlier
this year. Version 4.6 primarily introduces support for Data Standard 6 and
ODS/API 7.3.1, with no major user-facing changes for those building data models
in MetaEd. See the [release notes](/reference/metaed/releases/4.6.0) for MetaEd
4.6.
