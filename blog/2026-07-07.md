We’re pleased to announce that Certification 2.1.0 is complete and available in Ed-Fi Certification Suite v2.1.0.

This release delivers:

1. Completed Data Standard 5 certification coverage across the targeted certification scenarios
2. Completed AdminApp integration support for certification use
3. Stability and quality fixes based on regression findings and validation updates


<!-- truncate -->


1) Certification 2.1.0 Completed
Certification 2.1.0 scope is now fully delivered in this release, including DS5 certification validation and documentation updates for:

* Student
* StudentAcademicRecord
* StudentSchoolAssociation
* StudentContactAssociation
* Contact
* DisciplineAction
* StudentContactAssociation
* StaffSectionAssociation
* StudentSchoolAssociation
* StaffAssociation
* GradingPeriods
* StudentProgramAssociation
* StudentEdOrgResponsibilityAssociation

> These updates align certification behavior and validations with Data Standard 5 requirements for the 2.1.0 milestone.

2) AdminApp Integration Support Delivered

* AdminApp certification support for this release is complete
* Documentation and release artifacts are available for v2.1.0 consumers

3) Quality and Reliability Updates
* Addressed scenario validation issues identified during regression testing (v4 and v5)
* Refined assertions and naming consistency across updated DS5 scenarios
* Updated dependencies, including axios v1.15.0 (critical security patches)
